# Kubernetes Beginner to Pro Guide

Hands-on learning repository to master Kubernetes step by step through practical exercises.

## Learning Phases

- [ ] Phase 1: Foundations (clusters, pods, kubectl basics)
- [ ] Phase 2: Workloads (deployments, replica sets, services)
- [ ] Phase 3: Config & State (configmaps, secrets, volumes, pv/pvc)
- [ ] Phase 4: Networking (dns, ingress, network policies)
- [ ] Phase 5: Reliability (probes, resources, autoscaling)
- [ ] Phase 6: Security (rbac, service accounts, pod security)
- [ ] Phase 7: Observability (logs, metrics, troubleshooting)
- [ ] Phase 8: Production (helm, ci/cd, gitops)
